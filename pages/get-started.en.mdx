## Development

NotionNext is developed based on Next.js, and it is very easy to develop locally. First, Clone project on Github  [NotionNext](https://github.com/tangly1024/NotionNext.git)。

1. Install Next.js : `yarn add next`

2. Development : `yarn run dev`

3. Build a production package : `yarn run build`

4. Run on server ：  `yarn run start`


## Deploy and preview on Vercel

1. Install Vercel ： `yarn add vercel`

2. Preview the site  : `vercel`

3. Publish website  ： `vercel --prod`
