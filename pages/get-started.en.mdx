# Get Started

## Quick Start with Vercel

You can start by create your own Notion site and deploy to Vercel by clicking the link:

[![](https://vercel.com/button)](https://vercel.com/new/clone?demo-description=Notion%20powered%20site%20Built%20with%20Next.js.&demo-image=https%3A%2F%2Fdocs.tangly1024.com%2Fdemo.png&demo-title=Notion%20Next20Blog&demo-url=https%3A%2F%2Fblog.tangly1024.com%2F&project-name=notion-next-blog&repository-name=NotionNexts&s=https%3A%2F%2Fgithub.com%2Ftangly1024%2FNotionNext
)

Vercel will create the NotionNext repository and deploy the site for you with just a few clicks.  
Once done, every change in the repository will be deployed automatically.


## 手动创建 

NotionNext is developed based on Next.js, and it is very easy to develop locally. First, Clone project on Github  [NotionNext](https://github.com/tangly1024/NotionNext.git)。

1. 本地安装 Next.js : `yarn add next`

2. 开发调试 : `yarn run dev`

3. 编译打包 : `yarn run build`

4. 生产环境运行 ：  `yarn run start`


## 在vercel环境中预览与部署

1. Install Vercel ： `yarn add vercel`

2. Preview the site  : `vercel`

3. Publish website  ： `vercel --prod`
